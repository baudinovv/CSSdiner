### Technologies
* SCSS
* Vite
* HTML5
* JavaScript

## Support
* Only Dekstop version 

## Build Setup

Make sure that [pnpm](https://pnpm.io/) package is installed.

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
